# 🎮 Simon Game

A browser-based implementation of the classic **Simon memory game** — built with vanilla HTML, CSS, and JavaScript.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

## 🕹️ How to Play

1. Watch the sequence of colored button flashes
2. Repeat the sequence by clicking the buttons in the same order
3. Each round adds one more step — how far can you go?

## ✨ Features

- Progressive difficulty — sequence grows each round
- Visual & audio feedback on button press
- Game over detection with restart option
- Fully responsive design

## 📦 Run Locally

```bash
git clone https://github.com/Dearabhay/simon-game.git
cd simon-game
# Open index.html in your browser
open index.html
```

No dependencies or build step needed — pure vanilla JS!
